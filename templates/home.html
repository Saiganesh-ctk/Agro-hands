An Interactive Web Application For Farmer

We aim to provide a helping hand to users practicing farming resulting in sustainable agricultural growth and development. There are 3 modules in our work used for crop yield prediction and crop disease prediction. We have used remote sensing satellite imagery, Machine learning, and deep learning concepts,enabling better-informed decision-making in agricultural management practices.

Crop yield prediction

This module uses a tabular dataset that takes into account the amount of fertilizer, pesticide usage, area, production, and season to predict the crop yield. Cross validation and feature randomization techniques have been implemented to improve the model accuracy.

Crop yield prediction using remote sensing data

The decision-making capability is improved in this module with the help of remote sensing data. By integrating remote sensing data with meteorological data and building an ML model, the model gains a more comprehensive understanding of the factors influencing crop yield. This improved the accuracy of our model.

Leaf disease prediction

This module uses the leaf image of a plant as input to predict the crop disease if it is affected. The VGG-16 CNN model is used in this module. This deep learning technique has increased the model's accuracy.
